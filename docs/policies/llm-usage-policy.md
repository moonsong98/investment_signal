# LLM Usage Policy

LLMs are used only after rule-based filtering.

Allowed uses:
- Summarize important events
- Explain possible drivers and risks
- Draft research notes
- Generate open questions for human review
- Classify event context

Disallowed uses:
- Final buy or sell decisions
- Live order generation
- Position sizing decisions for real capital
- Publishing unreviewed claims as facts

Cost controls:
- Level 3 events only
- Daily and monthly request limits
- Token limits
- Prompt templates with concise context
- Mock LLM responses in local tests
