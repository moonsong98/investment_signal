# Do Not Build Yet

These are explicitly out of scope until a later approved milestone:

- Live broker API order placement
- Real account synchronization
- Real position size publication
- LLM-generated buy or sell decisions
- LLM-generated real order instructions
- High-frequency processing
- Complex production database infrastructure
- Unfiltered LLM analysis for every event

v0 through v3 must remain research, alerting, logging, and note-generation only.
